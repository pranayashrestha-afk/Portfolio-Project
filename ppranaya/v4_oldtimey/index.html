About This Portfolio

The website you are reading was built entirely by hand as the final project for CS427 at Kavya School. It required a minimum of four web pages demonstrating inline, internal, and external CSS techniques.

The project also demanded thorough written documentation — including wireframe design, testing records, and a deployment report. Every section of this portfolio was considered carefully in terms of both content and visual presentation.

Editor's Note All code in this portfolio was written by the student named above. No templates, Bootstrap, Tailwind, or JavaScript libraries were used at any stage of development.
